使用帮助

属性解释

drwxr-xr-x
一共十个字符,分别有他们的含义

  • [1 ] 档案类型
  • [2,3,4] 档案拥有者的属性
  • [5,6,7] 档案所在组的属性
  • [8,9,0] 其他人的属性
档案类型的值的解释
  • [-] 文件
  • [d] 文件夹
  • [l] link文件
  • [b] 则表示为装置文件里面的可供储存的接口设备
  • [c] 则表示为装置文件里面的串行端口设备，例如键盘、鼠标。
属性的值的解释
  • [-] 无此属性
  • [r] (read contents in directory): 表示具有读取目录结构清单的权限，所以当您具有读取 (r) 一个目录的权限时， 您就可以利用 ls 这个指令将该目录的内容列表显示出来!
  • [w] 建立新的档案与目录; 除已经存在的档案与目录(不论该档案是属于谁的!);将已存在的档案或目录进行更名; 移该目录内的档案、目录位置。
  • [x] 可执行(用户对当前文件夹没有x[可执行]的权限是,即使有r[可读]权限也无法访问该目录)
改变属性
  • chgrp 改变所属群组
  • chown 改变拥有者
  • chmod 改变档案的属性,suid

chgrp改变所属群组

[-R]属性表示递归,改变该目录下所有的文件与目录,都改成相同的权限

注意 改变到的那个群组,必须要在 /etc/group 文件中存在

chown改变拥有者

[-R]属性表示递归,改变该目录下所有的文件与目录,都改成相同的权限

注意 改变到的那个用户,必须要在 /etc/passwd 文件中存在

chmod改变文档属性
